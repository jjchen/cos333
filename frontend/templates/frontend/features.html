An introduction

Looking for something to do?

See what's happening around your area in a map, calendar or list view. Use the sidebar to filter by tags (green boxes), groups (red boxes), or collections (grey boxes). If you prefer to search for a specific title, location, or tag, you can do that as well.

Fight listserv spam

If you want to watch Game of Thrones with others, skip the e-mail and make an event on OC. Missed dinner? Find the free food event or study break closest to you by clicking "food" and checking the map view - no need to comb through "FreeFood" e-mails.

Plan events with friends

Simply check "Private" in the Make view, select the group you want to restrict this event to, and hit Save to ensure that only they can see it.

Facebook integration

Facebook integration allows you to easily import friends and groups from Facebook and export any events you have created in OC for easy cross-posting.

Go global

While the map defaults to Princeton, the default location can easily be changed within the Settings view - use OC around the world.
